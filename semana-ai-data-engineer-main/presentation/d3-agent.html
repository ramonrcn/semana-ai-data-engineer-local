TIPOS DE WORKFLOW

4 Padrões de Orquestração

Sequential

A → B → C

Cada agente executa em ordem fixa — Analyst → Researcher → Reporter

Parallel

A | B | C → merge

Múltiplos agentes executam ao mesmo tempo — resultado consolidado no final

Hierarchical

Manager → Workers

Um planner central delega tarefas — specialists executam e reportam

Consensus

A + B + C → Vote

Agentes independentes votam — a melhor resposta vence por consenso

O ShopAgent usa Sequential no Dia 4: Analyst analisa, Researcher pesquisa, Reporter sintetiza. Mas entender os 4 padrões permite projetar qualquer sistema.

→500 lines / day
